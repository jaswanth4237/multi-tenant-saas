# Product Requirements Document

## User Personas
### Super Admin
Manages all tenants

### Tenant Admin
Manages users & projects inside tenant

### User
Works on tasks

## Functional Requirements
FR-001 The system shall allow tenant registration  
FR-002 The system shall isolate tenant data  
FR-003 The system shall enforce subscription limits  

## Non-Functional Requirements
NFR-001 API response < 200ms  
NFR-002 JWT expiry 24h
